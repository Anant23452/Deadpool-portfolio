## 🚀 Installation

1. Clone the repository:
```bash
git clone https://github.com/Anant23452/Deadpool-portfolio.git
cd Deadpool-portfolio
```

2. Open in browser:
```bash
# Simple method - open index.html
open index.html

# Or use a local server
python -m http.server 8000
# Visit http://localhost:8000
```
